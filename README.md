## TickFocus
TickFocus is time- and task-tracking web application powered by Java+Spring

### Technology Stack

**Core:**
- Java 17
- Lombok
- Spring Boot 3.3.1

**Business layer**:
- Spring MVC
- Spring Security
- Hibernate Bean Validation

**Presentation layer**:
- Thymeleaf
- - HTML
- - CSS/SCSS
- - JS

**Persistence layer**:
- Spring Data JPA

**Database layer**:
- H2 database (for development)
- Postgres (for production)
